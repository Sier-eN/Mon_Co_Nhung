Giới thiệu chung

Mình hiện là sinh viên chuyên ngành Công nghệ thông tin với niềm đam mê lớn dành cho phát triển phần mềm và quản trị cơ sở dữ liệu. Mình luôn tìm tòi những cách tối ưu hóa hệ thống và xây dựng những ứng dụng mang lại giá trị thực tiễn cho cộng đồng.

Kỹ năng chuyên môn

Ngôn ngữ lập trình

  • Java (Swing, IntelliJ, Eclipse)
  • SQL Server (SSMS)
  • PHP & MySQL
  • Big Data (Hadoop, Scala)

Công cụ & Quy trình

  • Phân tích thiết kế hệ thống (UML)
  • Quản lý cơ sở dữ liệu phân tán
  • Thiết kế mạch Arduino (Proteus)
  • Cấu hình server Minecraft

Dự án tiêu biểu

  • Quản lý Balo Cặp sách: Phát triển ứng dụng quản lý bằng Java và SQL Server với đầy đủ sơ đồ UML.
  • Ẩm thực Hà Nội: Xây dựng hệ thống quản trị nội dung web bằng PHP.
  • Hệ thống Big Data: Triển khai các tác vụ MapReduce và xử lý dữ liệu lớn.

Sở thích & Đam mê

Ngoài việc lập trình, mình có niềm đam mê đặc biệt với bàn phím cơ (đang sử dụng Langtu LT84), tối ưu hóa phần cứng máy tính và tham gia cộng đồng modded Minecraft.
